# Chapter Template Reference

This file documents the standard format for auto-generated chapters.

## Frontmatter Schema

```yaml
---
title: "Chapter Title"
generated: "2025-12-16T00:00:00Z"  # ISO 8601 timestamp
generator: "generator-name"         # Which generator created this
source_files:                       # Files used to generate this chapter
  - "docs/VISION.md"
  - "cortical/analysis/pagerank.py"
tags:                               # Searchable tags
  - algorithms
  - foundations
---
```

## Required Sections

### Overview
Brief 2-3 sentence summary of the chapter.

### Main Content
The substantive content of the chapter.

### See Also
Links to related chapters and source code.

### Source Files
List of files this chapter was generated from, with specific line numbers if applicable.

## Footer

Every chapter ends with:

```markdown
---

*This chapter is part of [The Cortical Chronicles](../README.md),
a self-documenting book generated by the Cortical Text Processor.*
```
