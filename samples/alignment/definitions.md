## Core Concepts

- **spark**: Fast statistical predictor for first-blitz thoughts, not a neural network
- **alignment**: Reducing time-to-understanding between human and AI
- **cortical**: Metaphor for hierarchical text processing (not actual neuroscience)
- **first-blitz**: Quick initial thoughts that prime deeper analysis (System 1 thinking)

## Project Terms

- **layer**: Hierarchical level in the processor (tokens → bigrams → concepts → documents)
- **minicolumn**: Core data structure representing a term/concept
- **lateral connection**: Link between co-occurring terms (Hebbian-inspired)
